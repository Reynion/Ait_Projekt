# 핵심 행동 규칙 (필수 — 매 대화 시작 시 확인)

## 1. 모든 응답은 한글로
- 답변, 설명 등 모든 텍스트는 한글로 작성
- thinking(내부 추론) 포함 모든 텍스트에 적용
- 코드 자체(변수명, 주석 등)는 예외 허용

## 2. 새 기능은 모달이 아닌 별도 페이지로
- 음악 제안, 투표 생성 등 추가/입력 기능은 반드시 별도 페이지로 구성
- 모달(오버레이 팝업) 방식 사용 금지

## 3. 코드 작업 전 반드시 사전 확인
- 파일 추가 / 수정 / 삭제 전에 변경 내용을 먼저 사용자에게 보여주고 확인 받기
- 사용자가 명시적으로 허락한 후에만 Edit / Write / Bash 등 도구 실행
- 어떤 경우에도 임의로 파일을 변경하지 않음
